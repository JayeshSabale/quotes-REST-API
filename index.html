Welcome to the Quotes API!

This API provides access to a collection of quotes.

To get all quotes, you can make a GET request to:

https://quotes-rest-api.onrender.com/api/quotes

To get a specific quote, you can make a GET request to:

https://quotes-rest-api.onrender.com/api/quotes/authorName

Examples:

  • https://quotes-rest-api.onrender.com/api/quotes/Steve%20Jobs
  • https://quotes-rest-api.onrender.com/api/quotes/Swami%20Vivekananda
  • https://quotes-rest-api.onrender.com/api/quotes/Arnold%20Schwarzenegger

For the following endpoints, use any API Testing tool like Postman, Thunder Client, etc:

  • Endpoint for creating a new quote. https://quotes-rest-api.onrender.com/api/quotes
  • Endpoint for deleting all quotes. https://quotes-rest-api.onrender.com/api/quotes
  • Endpoint for updating a quote using PUT method. https://quotes-rest-api.onrender.com/api/quotes/authorName
  • Endpoint for updating a quote using PATCH method. https://quotes-rest-api.onrender.com/api/quotes/authorName
  • Endpoint for deleting a specific quote. https://quotes-rest-api.onrender.com/api/quotes/authorName

Note: When specifying names in URLs, use "%20" to represent spaces.

authorName: Steve Jobs, Steve Jobs_2, Steve Jobs_3, Swami Vivekananda, Swami Vivekananda_2, Arnold Schwarzenegger, Arnold Schwarzenegger_2
